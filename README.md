# 📊 Data Analysis Project

## 📌 Project Overview

This project demonstrates an end-to-end **Data Analysis workflow**, starting from raw data to actionable insights. It includes data loading, cleaning, exploratory data analysis (EDA), SQL querying, dashboard creation, and final reporting.

The goal of this project is to showcase practical data analysis skills using industry-standard tools like **Python, SQL, and Power BI**.

---

## 🛠️ Tools & Technologies Used

* **Python** (Pandas, NumPy, Matplotlib, Seaborn)
* **MySQL Workbench**
* **Power BI**
* **Gamma (for PPT creation)**

---

## 📂 Project Workflow

### 1. Data Loading

* Imported dataset using Python (Pandas)
* Checked data structure and initial observations

### 2. Data Cleaning

* Handled missing values
* Removed duplicates
* Fixed data types
* Cleaned inconsistent data entries

### 3. Exploratory Data Analysis (EDA)

* Analyzed data distributions
* Identified trends and patterns
* Created visualizations (charts & graphs)
* Generated insights for business understanding

### 4. SQL Analysis

* Loaded cleaned data into MySQL Workbench
* Wrote SQL queries for:

  * Aggregations (SUM, AVG, COUNT)
  * Joins
  * Filtering and grouping
* Extracted meaningful insights from structured queries

### 5. Dashboard Creation (Power BI)

* Built an interactive dashboard
* Key KPIs and metrics visualization
* Filters and slicers for dynamic analysis
* Easy-to-understand business insights

### 6. Reporting & Presentation

* Created a detailed report summarizing findings
* Designed a professional presentation using Gamma
* Included key insights, visuals, and recommendations

## ▶️ How to Run the Project

1. **Clone the repository**

   ```bash
   git clone <your-repo-link>
   ```

2. **Run Python Scripts**

   * Install required libraries:

     ```bash
     pip install pandas numpy matplotlib seaborn
     ```
   * Run the notebook/script for EDA & cleaning

3. **SQL Setup**

   * Import dataset into MySQL Workbench
   * Run SQL queries from provided `.sql` file

4. **Power BI Dashboard**

   * Open `.pbix` file in Power BI Desktop
   * Refresh data if needed

5. **Presentation**

   * Open PPT (created using Gamma) for project summary
